# Roadmap

## Done

- [x] Menu bar integration with status indicator
- [x] Dashboard with tunnel list
- [x] Cloudflare API integration
- [x] Start/stop tunnels
- [x] Real-time log streaming
- [x] Log filtering by level
- [x] Tunnel detail view (connections, ingress rules)
- [x] New tunnel creation wizard
- [x] Settings panel
- [x] Keychain credential storage
- [x] Dark mode support
- [x] Keyboard shortcuts
- [x] macOS notifications
- [x] Auto-reconnect on failure
- [x] VoiceOver accessibility
- [x] Persistent log storage to disk (`~/.tunnelflare/logs/`)
- [x] Tunnel search & filtering in dashboard
- [x] Connection health monitoring
- [x] Log export (plain text)
- [x] Quick connect from menu bar (start/stop tunnels)

## Planned

### v1.1.0
- [ ] Tunnel favorites / pinning

### v1.2.0
- [ ] Multiple account support (currently: single API token, multiple organizations)
- [ ] Tunnel templates (save & reuse configs)
- [ ] Ingress rule editor
- [ ] Path-based routing support (e.g., `example.com/api` -> service A)

### v2.0.0
- [ ] Remote tunnel control (tunnelflare-daemon for cross-machine management)

### Future
- [ ] Log export (JSON, CSV formats)
- [ ] Tunnel sharing (export/import configs)
- [ ] Homebrew distribution
- [ ] Sparkle auto-updates
